下記の要件と構成案を元に、スライド資料を生成するためのプロンプトを生成してください。

# 要件

- スライドは Gemini の Canvas 機能を利用して作成する
- Google slides形式で出力する
- 下記構成内容に適切なデザイン案があれば、プロンプトに盛り込む
- プロンプトに渡す構成案は、AIが処理しやすいようにJSONで構造化する
- 画像にはインターネットから取得したものを利用しない

# 構成案

```md
# AIネイティブ時代のLT資料作成術

# はじめに：現状と課題

- 資料作成は面倒・・・
- AIにやらせたいがうまくいかない・・・
- 「なんかボヤっとした内容・・・」「ダサくね・・・？」

# 提案：役割分担とプロンプト改善

- LT資料こそAIで楽したい！
- そのためにはどうするか・・・

1. AIに渡す役割を分ける
2. プロンプトにこだわる

# 具体的な内容：役割を分ける

- LTの要素を「構成」「スライド」「台本」に分ける

# 具体的な内容：プロンプトにこだわる

- プロンプトを駆使して発表内容の構成案を練る
- 出力された構成案の内容をAIに渡して画像・台本を生成する

# でもプロンプト作成は難しい

- メタプロンプトを利用して、「プロンプト」自体を生成しよう

# 作例

(別のプレゼンテーション資料に移り、そちらを紹介)

# 実際のプロンプト紹介

（後でコードブロックとして入力）

# まとめ・結論

- メタプロンプトを使えばプロンプトにこだわれる！
- そうすれば抽象的な・ダサいプレゼン資料から脱却できる！
- 注意点
  - AI臭さを嫌うシチュエーションでは不適切カモ
  - 今回はコンプライアンスや著作権等を考慮してないです

# ご清聴ありがとうございました
```
